SYS · 06 / Honest scope

Deliberately small. Deliberately clear.

SQLite is a one-writer database

That is a good trade for a small fleet and one-file backups. It is not the storage plan for a high-concurrency national network.

No route-optimization AI

HubRoute records routes, assignments, custody, and events. It does not promise automated vehicle routing or demand prediction.

Vercel is a separate starter

The production path is PHP + local SQLite on shared hosting. The Vercel artifact uses hosted libSQL/Turso and is not a drop-in copy of that runtime.

SYS · 07 / FAQ

Questions before the first parcel.

Do I need terminal or SSH access?

No. Build or download the zip, upload it in cPanel File Manager, extract it, select PHP 8.1+, and open the health check.

How do backups work?

php maintenance.php run --apply backs up the SQLite database before pruning old terminal, audit, idempotency, and rate-limit rows. Schedule it with your host's cron panel.

Does HubRoute charge per parcel?

No. The software has no per-order platform fee. You still pay for your web host, domain, and any infrastructure or support you choose.

Can I use the Vercel starter for production?

It is a separate foundation for public tracking and hosted libSQL/Turso. Follow its setup and security guide; do not assume it is the same runtime as the full PHP operations console.

What should I do with the seed accounts?

Create a real production administrator, then rotate or disable every seeded account and confirm the published demo passwords no longer work.

Security

Small enough to audit, strict enough to run on a real host.

Hardened runtime

Security headers, secure session settings, idle and absolute timeouts, generic error output, and server-side logging.

Safer data access

PDO prepared statements, whitelisted status filters, escaped output, and authorization checks around parcels, routes, hubs, agents, and CSV export.

Shared-hosting guardrails

The data directory receives Apache denial files, and setup notes recommend moving SQLite storage outside the web root whenever possible.

Checklist

Before this becomes a production parcel console.

  • Rotate every seeded password, especially the walkthrough hub login shown on this page.
  • Keep `.env` private and move `DATA_DIR` outside `public_html` if your hosting plan allows it.
  • Confirm `data/hubroute.sqlite` is not downloadable after first run.
  • Enable HTTPS at the host or CDN and keep PHP on a supported 8.x release.
